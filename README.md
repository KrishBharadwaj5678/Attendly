# 📊 Attendly 

A simple and useful application that helps students **track their attendance percentage** and understand how many classes they can **miss or need to attend** to maintain the required minimum attendance.

![AttendlyDemo](https://github.com/KrishBharadwaj5678/Attendly/raw/main/AttendlyDemo.jpeg)

## ✨ Features

| Feature                      | Description                                        |
| ---------------------------- | -------------------------------------------------- |
| 📊 Attendance Percentage     | Calculates the current attendance percentage       |
| 🧮 Smart Calculation         | Automatically calculates attendance statistics     |
| ❌ Missed Classes             | Shows how many classes the user has already missed |
| 💤 Bunk Calculator           | Tells how many classes the user can safely bunk    |
| 🎯 Minimum Attendance Target | Allows users to set required minimum attendance    |
| 📱 Simple UI                 | Clean and easy-to-use interface                    |

## 🚀 Getting Started

### 1️⃣ Clone the repository

```bash
git clone https://github.com/KrishBharadwaj5678/Attendly.git
```

### 2️⃣ Navigate to project directory

```bash
cd Attendly
```

### 3️⃣ Install dependencies

```bash
flutter pub get
```

### 4️⃣ Run the app

```bash
flutter run
```
